# Sales-dashboard-powerbi
Sales Performance and Profit Optimization Dashboard using Power BI

## Project Overview
This project analyzes sales performance, profitability, and regional trends using Power BI. The dashboard helps identify key business insights such as high-performing categories, loss-making segments, and regional sales distribution.

## Key Features
* KPI metrics: Total Sales, Profit, Profit Margin %, Avg Order Value
* Category-wise and region-wise performance analysis
* Identification of loss-making sub-categories
* Interactive filters for dynamic data exploration

## Key Insights
* Technology category contributes highest profit (~50%)
* Furniture shows lower profitability despite sales
* West region leads in quantity but not in profit margin
* Discounts negatively impact overall profit

## Tools Used
* Power BI
* DAX
* Data Visualization
